# UI Components

Shared UI component library built with shadcn/ui and Tailwind CSS.

## Components

- Button
- Input
- Textarea
- Select
- Card
- Tabs
- ScrollArea
- Label
- Slider

## Usage

```tsx
import { Button, Input, Card } from "ui-components";

function MyComponent() {
  return (
    <Card>
      <Input placeholder="Enter text" />
      <Button>Click me</Button>
    </Card>
  );
}
```
